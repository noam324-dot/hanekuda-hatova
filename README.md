# הנקודה הטובה

אפליקציית Next.js בעברית, מותאמת לנייד ול־RTL. כל המידע האישי נשמר בדפדפן בלבד באמצעות `localStorage`; אין שרת, מסד נתונים או הרשמה.

## פיתוח מקומי

```bash
npm install
npm run dev
```

## בנייה והרצה

```bash
npm run build
npm start
```

## פריסה ב־Vercel

ייבוא המאגר ל־Vercel מספיק. המערכת תזהה את Next.js ותשתמש אוטומטית ב־`next build`. אין צורך במשתני סביבה או בשירותים חיצוניים.
